# geneatree-server

Persistence layer for [geneatree](https://github.com/thoughtsunificator/geneatree).

## Getting started

### Run

- ```npm install```
- ```npm start```

## Events

See [Persistence](https://github.com/thoughtsunificator/geneatree/persistence#events)

Supported events : 

- individualNotesAdd
- individualNotesUpdate
- individualNotesRemove
- individualRemove
- individualAdd
- individualUpdate
- treeAdd
- treeUpdate
- treeRemove
